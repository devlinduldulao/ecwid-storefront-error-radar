Storefront Error Radar Support

Use this page as the support URL for the published Ecwid app. This repo uses GitHub Issues as the primary support intake and keeps product documentation in the public repository.

Support channels

Issue tracker: github.com/devlinduldulao/ecwid-storefront-error-radar/issues

Repository: github.com/devlinduldulao/ecwid-storefront-error-radar

Suggested hosted app URL: devlinduldulao.github.io/ecwid-storefront-error-radar/public/index.html

Diagnostic workflow

Ask merchants to open the app inside Ecwid admin, reproduce the issue in the embedded preview, and export the JSON session. Because this app is owner-only and browser-local, support should focus on merchant reproduction steps instead of live visitor history. Include the store ID, the page path that failed, and the exported JSON file in the GitHub issue.
