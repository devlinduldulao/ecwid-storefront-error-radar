Privacy

Storefront Error Radar Privacy Notice

This Ecwid app is an owner-only diagnostics tool. It records runtime issues only while the merchant uses the embedded preview and keeps that data browser-local by default.

What this app stores

The app stores browser-local settings such as capture preferences, demo-preview mode, the max event cap, and the slow-request threshold. Session incidents exist only in the current browser unless the merchant exports them manually.

What this app does not collect

This version does not run for live shoppers, does not send telemetry to a backend controlled by this repo, and does not create a centralized customer event archive. It avoids databases, Redis, and webhook ingestion on purpose.

Exported files

When a merchant clicks Export JSON, the file is generated in the browser and downloaded locally. Review exported files before sharing them outside your support process.

Support contact

Support requests for this app are handled through the public issue tracker at github.com/devlinduldulao/ecwid-storefront-error-radar/issues. Do not include secrets in exported JSON files or support tickets.
